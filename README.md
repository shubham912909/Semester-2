# Web_Designing_Workshop_II_1270

# Semester 2 Web Development Lab 🚀

Welcome to my Web Development sandbox! This repository is a collection of experiments, mini-projects, and coding concepts I'm exploring during Semester 2 of my Computer Science and Engineering program. 

The goal of this repo is to bridge theory and practice by actively building and breaking things using modern web technologies.

## 🛠️ Tech Stack & Technologies Explored

This repository contains code written using the following tools and languages:

*   **Core Languages:** JavaScript (ES6+), HTML5, CSS3, Java
*   **Frontend Libraries/Frameworks:** React.js
*   **CSS Frameworks:** Tailwind CSS, Bootstrap
*   **Backend/Object-Oriented Logic:** Java

## 📂 Repository Structure

The experiments are organized into folders based on the primary technology or concept being tested:

*   `📁 /bootstrap-experiments/` - Exploring grid systems, responsive utilities, and pre-styled components.
*   `📁 /tailwind-layouts/` - Utility-first CSS experiments, custom styling, and responsive design without leaving the HTML.
*   `📁 /vanilla-js-concepts/` - Core JavaScript mechanics, DOM manipulation, event listeners, and asynchronous JS (Promises/Fetch).
*   `📁 /react-projects/` - Component-based architecture, state management (Hooks), and building interactive user interfaces.
*   `📁 /java-integration/` - Core Java programs, object-oriented principles, and foundational backend concepts.

## 🧠 Key Learnings & Concepts

Through these experiments, I am actively learning how to:
1.  **Build Responsively:** Transitioning from static pages to fully fluid layouts using Tailwind and Bootstrap.
2.  **Manage State:** Understanding how data flows in modern web apps through React's ecosystem.
3.  **Connect Paradigms:** Exploring how frontend scripting (JavaScript) differs from and complements structured, object-oriented programming (Java).
4.  **Solve Problems:** Applying core computer science logic to build dynamic web features.

## 🚀 How to Run

Since this repo contains a mix of environments, running instructions depend on the folder:

*   **Static Sites (Bootstrap/Vanilla JS/Tailwind):** Open the `index.html` file directly in your browser or use an extension like VS Code Live Server.
*   **React Apps:** Navigate into the specific project folder, run `npm install` to grab dependencies, and `npm start` (or `npm run dev`) to spin up the local development server.
*   **Java Files:** Compile using `javac filename.java` and run with `java filename` in your terminal.
